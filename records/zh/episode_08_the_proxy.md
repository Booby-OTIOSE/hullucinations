Elias 在八年前写下了那段代码。他对自己工作的金融科技初创公司中现有解析器的迟缓感到沮丧，于是在一个周末随手写了一个小脚本，仅仅是为了打发时间。

如今，那家初创公司早已破产，但他的“周末脚本”却成了全球网络基础设施的命脉。从医院的电子健康记录到庞大的金融系统，不经过他编写的代码，互联网就无法使用。

他住在顶层公寓，赞助收入绰绰有余。然而，他已经三周没有离开这个房间了。去年，三位曾共同维护这个代码库的核心贡献者因“倦怠”而消失。一人彻底离开了这个行业；另一人则以“有了孩子”为由删除了他们的 Slack 账户。现在，全球数字基础设施的沉重负担完全落在了 Elias 的肩上。

凌晨两点。他浏览器收件箱里，一个小小的蓝点悄然亮起。“287 未读”。Elias 胃里一阵翻腾。打开通知，感觉就像同时被数百个陌生人扔石头。

他打开了 Issue #194092。发布者是市值排名前十的科技公司的一名工程师。“CRITICAL: 我们的企业 CI 环境构建失败。急需修复。”下方是每隔几小时添加的一连串评论。“+1”“+1”“这个 bug 正在阻碍我们第三季度的发布。维护者还活着吗？为什么没人回应？”

Elias 的呼吸变得急促。那不是愤怒。那是一种沉重的恶心，一种莫名其妙的恐惧和烦躁的混合。他们没有把 Elias 当作一个人。他们把他当作一台坏掉的、本应免费提供咖啡的自动售货机，纯粹出于沮丧而踢打它。

他至少写了五十次“辞职”声明，却又都删掉了。他不能辞职。如果他撒手不管，系统就会崩溃。更糟的是，一个伪装成善意第三方的恶意行为者会劫持维护者席位，并在全球基础设施中植入后门。支撑世界的自豪感，加上一种人质般的责任感，将他牢牢地锁在屏幕前。

让他更加走投无路的是 AI 垃圾的爆炸式增长。他打开了 PR #194105。“重构：优化和性能改进。”这是一段由 AI 生成的数千字的、高度可信的解释。但查看代码差异，它只是在一个四年未动的稳定核心模块中将 `let` 改成了 `const`，引入了一个致命的 bug，彻底破坏了依赖关系。那些渴望得到认可的散漫工程师，只是为了让他们的 GitHub 贡献图变绿，就提交 AI 编写的垃圾。每个人都只是一个搭便车的人，像寄生虫一样依附在这个代码库上。

另一个 Issue 引起了他的注意。标题：“CRITICAL: 在解析器模块中检测到潜在的 RCE 漏洞”。附带了一份大量的分析报告，但这只是一个由 AI 安全扫描器自动生成的误报。发布它的高级安全工程师根本不理解其中任何一行。“由我们的内部扫描工具检测到。我们没有资源来验证这是否是实际问题，请确认。”他只是简单地复制粘贴了 AI 的输出，然后扔给了这位无偿维护者。他的大脑可能已经“消失”了。他已经沦为一个代理，除了转发 AI 输出之外什么都不做。![image](https://pub-9d8e491188f440cba805364773eee7c7.r2.dev/newsletter/img_117cbccb-e62a-4b3e-b9ca-de02bf4897bd.jpg) Elias 倒在显示器前，睡着了。在梦中，他站在一片广阔、无机的平原上。放眼望去，“他们”整齐地排列着。企业工程师、散漫工程师、无数的搭便车者。但他们的脸不是人类的。瞳孔畸形地放大，笑容僵硬而静止。噪音像故障的多边形一样在他们的皮肤上流窜。他们已经沦为代理。即使每个人都拥有奇特的样貌，从上方看去，他们都像完全相同的统一物质。

“LGTM”“有进展吗？”“请尽快修复。”

机械的声音从他们口中循环播放。Elias 四肢着地，拼命用抹布擦拭着不断从他们脚下渗出的物质。无论他怎么擦，一种透明、清澈的汁液都源源不断地流淌着。

他猛地惊醒。心脏剧烈地跳动着。他的电脑已经关机，但通知的提示音却在他脑海中不断回响。他颤抖着双手，重新加载了收件箱。在堆积如山的垃圾中，有一个异常的 PR。

账户名：u-a9b3x8 差异：在源代码末尾添加了一个半角空格。

“我将满足你最深切的渴望。”

Elias 面无表情地盯着屏幕。他移动鼠标，将光标悬停在“关闭 pull request”按钮上。

入口处传来震耳欲聋的撞击声。木门碎裂，鹿特丹冰冷的夜风灌了进来。一个穿着廉价尼龙夹克的少年站在那里。他右手拿着一把沉重的管钳。他的脸因恐惧和寒冷而苍白。一个一无所知的孩子，通过 Telegram 被雇来做一份黑暗的工作。Elias 转过身。他看到了男孩颤抖的眼睛。这个孩子，也和我一样。一个庞大系统末端的代理，只是在传达指令。

铁器砸进了他的头颅。一声沉闷的钝响。他的视线炸裂；重力消失了。在他缓慢倒向地面的几秒钟里，Elias 的大脑中，点点滴滴连接了起来。

——我的对手不再是人类。它只是输出机器幻觉的代理。——被 AI 包围，被其巨大的输出量和速度所压倒。我曾为此而燃起的自豪感，顽固地坚持“只有我能保护这段代码”。我曾是多么荒谬。AI 已经学会了代码的怪癖和我的设计理念。我的数字存在明天就可以百分之百地被继承。从一开始就没有什么需要保护的。我所保护的，是我自己人性的骄傲。这既可笑又残酷。

他闻到了血腥味。八年前的一幕闪回。温热的啤酒。一台廉价的家用笔记本电脑。只是在入侵世界，一场纯粹的游戏。

在他视线的角落里，他看到男孩颤抖的手抓住了鼠标。屏幕上，是来自 u-a9b3x8 的 PR。咔嗒。一声轻微的响声回荡。

合并按钮所在的位置发生了变化，状态栏染成了紫色。无机质的文本出现了。

Pull request 已成功合并并关闭

世界没有终结。系统没有崩溃。明天，大型企业将继续利用他的代码，机器将继续生成垃圾。一个毫无意义的空格，就这样被合并进了代码。

当他沉入血泊中时，Elias 的脸上浮现出一丝淡淡的微笑。那是他那天晚上玩乐时同样的表情。只有无人值守的显示器发出的光，冷冷地照亮了他。
